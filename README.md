# Financial Fraud Detection & Predictive Maintenance Projects

This repository contains two essential data science projects connecting Machine Learning with high-impact industries: **Finance (FinTech)** and **Aviation (Logistics)**.

---

## Project 1: Credit Card Fraud Detection (Finance)
- **Concepts Applied**: Imbalanced Data Management, Random Forest Classifier, Threshold Optimization.
- **Results**: Achieved a **Recall of 0.96 (96%)** for fraud detection, maximizing financial protection while minimizing false card blocks.

---

## Project 2: Aircraft Engine Predictive Maintenance (Aviation)
- **Concepts Applied**: Time-Series Sensor Analysis, Rule-Based Triggers, Financial Risk Modeling.
- **Results**: Identified the optimal grounding window before engine failure, proving preventive maintenance saves **$450,000** per engine.

---
